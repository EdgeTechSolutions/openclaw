## Technical Learnings

### Confluence CLI markdown converter fix
- `marked` v3+ breaks the old renderer API — passes objects instead of strings to `renderer.code()` and `renderer.table()`
- Fix: check `typeof code === 'object'` and extract `.text`/`.lang` for code, and `.header`/`.rows` for tables
- Converter at `/workspace/tmp/md-to-confluence.cjs` — now handles both v2 and v3 APIs

### Von Neumann file transfer pattern (NEW)
- `sessions_send` has message size limits — base64 of files >10KB gets truncated
- **Correct pattern**: von Neumann saves locally → uploads to Litterbox → sends URL via sessions_send
- Von Neumann keeps files organized under `/workspace/projects/<project-name>/`
- `sessions_send` is now enabled for von-neumann
