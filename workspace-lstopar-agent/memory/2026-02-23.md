### OpenClaw Tool Policy Layers (learned)
- **Global**: `tools.alsoAllow` — applies to host sessions
- **Per-agent**: `agents.list[].tools` — overrides global for that agent
- **Sandbox**: `agents.defaults.sandbox.tools` — overrides for sandboxed sessions
- Sandbox sessions only see tools allowed at the **sandbox level**, not just global
